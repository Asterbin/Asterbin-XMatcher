1. Local API / 本地服务

Connects the page to the local XMatcher backend. / 连接本机 XMatcher 后端。

Run first in the project root / 先在项目根目录运行：
python xmatcher_local_api.py --database MP500_xrd_database.pkl

2. Upload XRD / 上传数据

Supports CSV/TXT/XY/DAT, simple JSON, and common XRDML/XML. The first two numeric columns are read as 2θ and intensity.
支持 CSV/TXT/XY/DAT、简单 JSON、常见 XRDML/XML。默认取前两列数值作为 2θ 和强度。

3. Search Scope / 检索范围

Use element filters to reduce false positives and speed up matching. / 用元素过滤可减少误匹配并加快检索。

Leave blank for full database search. Use comma or space separators. / 留空表示全库检索，可用逗号或空格分隔。

Number of returned matches. / 返回的候选物相数量。

4. Peak Detection / 寻峰参数

Controls preprocessing and experimental peak selection. / 控制预处理和实验峰筛选。

Ignore weak peaks below this relative height. / 低于该相对高度的弱峰会被忽略。

Higher values keep only more isolated peaks. / 值越大，越偏向保留突出峰。

Minimum 2θ spacing between detected peaks. / 相邻实验峰的最小 2θ 间距。

Top experimental peaks used for matching. / 用于匹配的最强实验峰数量。

Odd window size for denoising. / 去噪平滑窗口，建议使用奇数。

Fraction used for baseline correction. / 用于背景扣除的窗口比例。

5. Matching / 匹配参数

Controls peak alignment, score weighting, and minimum evidence for a phase. / 控制峰位校准、打分权重和候选物相的最低匹配证据。

Allowed 2θ error after shift correction. / 平移校正后的允许 2θ 误差。

Reject phases with fewer matched peaks. / 匹配峰数不足的候选会被排除。

Maximum global 2θ correction searched. / 搜索的最大整体 2θ 平移量。

Smaller is finer but slower. / 越小越精细，但计算更慢。

Weight for 2θ agreement. / 峰位一致性的打分权重。

Weight for relative intensity agreement. / 相对强度一致性的打分权重。

Hybrid is recommended for routine use. / 常规识别建议使用 hybrid。
